【中】余华《第七天》

发表于 2024-02-02 更新于 2024-02-03 分类于 文学
本文字数： 1.1k 阅读时长 ≈ 1 分钟

死亡、世界的参差、现实主义、黑色幽默

开篇

文章小引采取了类似《窄门》的开篇，节选圣经片段“到第七日，神造物的工已经完毕，就在第七日歇了他一切的工”，直接阐明主题：死亡（中华文化的传统习俗和近现代遭受的苦难其实常常让人们对死亡这件事三缄其口），而以余华风格，自然要添上多条看似散乱却实则相当有条理的故事线，再辅以相当的黑色幽默故事来将这个悲剧故事娓娓道来。

设定

人们在离世后需要火化，而灵魂只有在墓地之中才能安息，如果人们离世时没有自己的墓地，将会去往“死无葬身之地”。

细节（仅为个人理解）

“我”——杨飞，被爆炸带走了生命（文章开篇描写风格类似彼得汉德克，从“他者角度”来描述自身状态，并将动作、状态细分化），将以灵魂形式探寻所有谜团，而文章就在这一片淡淡的悲伤氛围中平静地展开。

小时候作为从火车卫生间“不小心”被母亲丢在铁轨上的孩子，被善良的杨金彪领养，在养父和他同事李月珍一家的呵护下成长，期间父亲放弃了本能结婚的机会选择了“我”，而我也在生母被认出并短暂的相处后，选择回家陪伴养父（生母一家的认亲桥段莫名像不久之前的那个寻亲成功的新闻，颇有搏人眼球，胡乱炒作的意味）。随后，在工作时结识了公司最为漂亮的同事李青并结婚，甜蜜的婚姻生活并没有一直维持下去，婚后李青为了寻求自己的理想，选择离婚和最懂自己的人在一起，而在之后不久，父亲身体情况急转直下，“我”变卖家产一路寻找父亲。在出租屋中结识了邻居“鼠妹”李梅和伍超，在第一次约好去给小敏补课赚钱的途中，遇到了房屋被强拆而失去双亲的小敏（此时“我”并不知道她父母已经离世），由于没有认出，去到了谭家鑫的餐馆去吃午饭，就在这时，一场爆炸发生了，而“我”由于被报纸上李青去世的新闻吸引也没能逃出餐馆。

七天对应了文中主要的七个事件：

1.我杨飞和父亲杨金彪的事件

2.警察张刚和李姓男子的事件

3.妻子李青的事件

4.养母李月珍和医疗事故造成的27个死婴的事件

5.小敏父母和因火灾被瞒报的38个人的事件（均为篇幅较短，算为一部分）

6.“鼠妹”李梅和伍超的事件

7.餐馆老板谭家鑫的事件

个人感觉比较好的段落

1.我在情感上的愚钝就像是门窗紧闭的屋子，虽然爱情的脚步在屋前走过来又走过去，我也听到了，可是我觉得那是路过的脚步，那是走向别人的脚步。

2.我走出自己趋向繁复的记忆，如同走出层峦叠翠的森林。疲惫的思维躺下休息了，身体仍然向前行走，走在无边无际的混沌和无声无息的空虚里。空中没有鸟儿飞翔，水中没有鱼儿游弋，大地没有万物生长。

**3.我对他说：“走过去吧，那里树叶会向你招手，石头会向你微笑，河水会向你问候。那里没有贫贱也没有富贵，没有悲伤也没有疼痛，没有仇也没有恨……那里人人死而平等”

他问：“那是什么地方？”

我说：“死无葬身之地。**

电吉他入门到放弃

发表于 2023-11-30 更新于 2023-12-07 分类于 音乐
本文字数： 820 阅读时长 ≈ 1 分钟

以下所有观点以及学习方式均为本人自学所用，仅从个人学习历程出发，无法保证正规性

方法论

作为没有童子功的普通人， 电吉他学习应当和其他乐器相同，喜欢什么就先学什么，先去找十几个喜欢的电吉他曲目，找到其中最简单的曲目开始坚持练习，在不断练习、弹奏的过程中，锻炼演奏技巧和乐理知识。从兴趣出发坚持练习才是最有效的学习方法。

这里推荐：一鸣老师、麦口罩老师

硬件入门

依据想要弹奏的曲目，电吉他需要使用不同的效果器、箱头，可以购买相应模块。这个部分稍微有点复杂，可按个人需求进行购买。

吉他主体

个人使用Yamaha的入门款PAC112，推荐Ibanez的入门款。
一般上面会有摇把、音色旋钮、音量旋钮、拾音器组合调钮，四种可以调节的部分，建议把音色、音量拉满，剩下的

效果器（按照连接顺序）

前级效果器：pedal类：音量踏板、表情踏板、哇音踏板（基本不用，独特发音）；compress压缩模块（使用较多，音色紧实）；EQ均衡（高手常用，不懂）；noise gate降噪门（常用，降低底噪）；boost激励（也叫推子，最常用，提升失真）；overdrive过载（常用，音色沙哑）；distortion失真（常用，音色尖锐）；fuzz法兹（失真的一种，万能青年旅店乐队有一些歌会用）

后级效果器：调制类：phase相位、flanger镶边、chrous合唱、tremolo颤音、reverse反向；复制类：delay延迟、reverb混响（这部分就全凭感觉来调）。

本人使用：NUX的MIGHTY PLUG PRO可以设定多种效果器以及箱头，而且很便携、可使用蓝牙进行连接。
